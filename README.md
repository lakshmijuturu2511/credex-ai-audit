# Credex AI Audit

A full-stack AI spend auditing platform built for startups and engineering teams to identify overspending across AI tools like ChatGPT, Claude, Cursor, Gemini, and Copilot.

The platform analyzes current AI tooling costs, generates optimization recommendations, estimates monthly and annual savings, and creates shareable audit reports with persistent storage using Supabase.

## Features

- AI tooling spend audit engine
- Savings recommendations and optimization logic
- Shareable public audit URLs
- Supabase backend integration
- Persistent audit storage
- Responsive modern UI
- GitHub Actions CI workflow
- Vercel deployment
- TypeScript + Next.js stack

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- Supabase
- Vercel
- GitHub Actions
